## Installation Instructions
1. Clone the repository:
   ```
   git clone [repository-url]
   ```
2. Navigate to the project directory:
   ```
   cd [project-directory]
   ```
3. Install the required Python libraries:
   ```
   pip install -r requirements.txt
   ```
   (Note: The `requirements.txt` file should list all necessary libraries and their versions.)

## Running the Application
1. To run the Jupyter notebook:
   ```
   jupyter notebook 'Sales forecasting project Final Notebook (1).ipynb'
   ```
2. To launch the dashboard:
   ```
   python final app.py
   ```
